# 📉 Async Web Price Tracker & Parser with Telegram Alerts

Асинхронный многопоточный парсер цен интернет-магазинов на Python. Позволяет мониторить цены на заданные товары в реальном времени и мгновенно отправляет оповещения в Telegram-канал или личные сообщения при их изменении.

---

## 🛠️ Технологический стек

* **Язык:** Python 3.10+
* **Асинхронность:** `asyncio` (неблокирующий опрос и планирование циклов)
* **Сетевой клиент:** `aiohttp` (асинхронный HTTP-клиент с пулом соединений для параллельных запросов)
* **Парсинг:** Регулярные выражения (Regex) (эффективное извлечение данных без накладных расходов тяжелых HTML-библиотек)
* **Конфигурация:** `python-dotenv` (управление токенами и настройками)
* **База данных:** Локальный JSON-сейвер для хранения истории цен.

---

## ⚡ Особенности архитектуры и асинхронности

Бот спроектирован с упором на производительность и неблокирующую обработку:
1. **Параллельный опрос (`asyncio.gather`):** Запросы ко всем страницам товаров выполняются параллельно. Бот не ждет ответа от одного сайта, чтобы опросить следующий, что экономит до 90% времени при сканировании сотен ссылок.
2. **Неблокирующие уведомления:** Отправка сообщений в Telegram вынесена в фоновый пул потоков (`loop.run_in_executor`), чтобы сетевые задержки Telegram API не тормозили процесс мониторинга.
3. **Эмуляция браузера:** Парсер автоматически добавляет правильные `User-Agent` заголовки для обхода базовой защиты от роботов на сайтах.

```
price-tracker-parser/
├── .env.example       # Шаблон конфигурационных данных
├── requirements.txt   # Список зависимостей
├── main.py            # Главный цикл мониторинга и логика алертов

```
